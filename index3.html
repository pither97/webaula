HTML

html e um codigo para descrever a estrutura de um codigo e tudo sobre a pagina a ser criada, define as tabelas e outros elementos por meio de (tags) que sao um tipo especifico de marcas.

Tags

Define um documento dentro do html e os tipos de tags sao: body(define o tipo de corpo do documento), h1 ate h6 servem para definir o titulo do documento,(p) para definir um paragrafo,(!---) define um tipo de comentario e esses paragrafos vao entre os signos de maios ou menos

HTML5

essa e a versao mais recebte do html e vem representada pela estrutura DOCTYPE html

Listas(ordenada e desordenada)

A lista ordenada vem representada pelo (ul) que mostra a lista de conteudo de forma numerica em ordem e tambem temos a (ol) que mostra o conteudo de tal forma que elas so teham uns pontos antes do paragrafo mas sem indicar mas sem indicar uma ordem.

Section

representada uma sesao dentro de um documento
